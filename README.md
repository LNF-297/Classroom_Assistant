# 微课堂
一款服务大学课堂的工具，提供签到、quiz、随机选人等功能。  
本程序为WHU弘毅学堂高级语言程序设计期末考试大作业，尚未通过稳定性和压力测试，尚未对非法连接及请求进行过滤，请勿用于生产环境。   
支持多种数据库，默认为基于ODBC的MySQL，可自行更改。  
采用多线程架构，可满足高并发场景。    
服务器和客户端采用socket文本透明传输，使用json格式，请参考APIDocument.md
